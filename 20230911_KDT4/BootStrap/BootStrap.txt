BootStrp
  2010년 Twitter의 Mart Otto, Jacob Thornton에 의해 Twitter Blueprint가 탄생.
  HackWeek 이라는 개발 이벤트 개최. 다수의 Twitter 개발자가 참가 오픈소스로 공개되기 전까지 사내 스타일가이드로 폭 넓게 사용됨.

  이후 'Twitter Blueprint'에서 'Twitter BootStrap' 이름으로 변경 2011년 08월 09일 오픈소스 프로젝트로 공개. 
  Mart Otto, Jacob Thornton 퇴사 후 Twitter에서 독립된 프로젝트가 되어 'Twitter BootStrap'에서 'BootStrap'으로 변경.


Bootstrap 구성 요소
  bootstrap은 레이아웃, 기본적으로는 CSS 요소를 위한 bootstrap.css 파일과 
  자바스크립트 플러그인 사용을 위한 bootstrap.bundle.js 파일이 필요하다.

  Responsive Web Desing - 잘 반응하는
    디바이스나 Web 브라우저 등 유저가 이용하는 환경에 맞추어 화면의 크기나 조작성을 변경.

  Fluid Grid (가변 그리드)
    브라우저 폭의 변화에 따라 내용의 크기를 변화시키는 레이아웃 수법

  Fluid Image
    레이아웃 폭에 따라 이미지의 사이즈를 확대, 축소하는 방법

  Media Queries
    CSS3에서 도입된 기술로 디바이스의 종류, 화면 넓이, 화면 해상도, 디바이스의 방향에 따라 스타일을 변경할 수 있다.
